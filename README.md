Katy Saintin – Personal Professional Website

Vision

Create a modern, elegant and maintainable personal website acting as a central hub for all professional and public activities of Katy Saintin.

The website should not be a traditional résumé.

It should be a professional portfolio showcasing:

* Scientific Software Engineering
* Scientific Data Engineering
* Control Systems (SCADA, EPICS, TANGO, MUSCADE)
* Artificial Intelligence
* Publications and Conferences
* Teaching and Knowledge Transfer
* Technical Leadership
* Outreach and Content Creation

The objective is to provide a complete digital identity while maintaining a clear separation between professional achievements and personal universes.

Architecture

The website follows a strict MVC-inspired architecture.

Model

Single source of truth:

data/data.json

Contains:

* site metadata
* menu
* hero
* statistics
* publications
* projects
* universes
* footer

⸻

Controller

js/site.js

Responsible for:

* loading JSON
* language management
* rendering pages
* Google Analytics

No business data should be hardcoded.

Views
